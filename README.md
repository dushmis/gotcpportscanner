gotcpportscanner
================

tcp port scanner in golang


Usage
=====

```./portscan -p=80 --host=google.com -w=1```

```./portscan --start=20 --end=9000 --host=google.com -w=1 -v ```

```./portscan --start=20 --end=9000 --host=google.com -w=1 -s```

```./portscan --p=22 --host=rasp```

```./portscan```
